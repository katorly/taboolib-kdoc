SkullUtils - Apply skull texture from different sources. Skull Meta: https://hub.spigotmc.org/javadocs/spigot/org/bukkit/inventory/meta/SkullMeta.html Mojang API: https://wiki.vg/Mojang_API

Some websites to get custom heads:

  • https://minecraft-heads.com/

The basic premise behind this API is that the final skull data is contained in a GameProfile either by ID, name or encoded textures URL property.
