isOneOf

open fun isOneOf(@Nullable materials: @Nullable Collection<String>): Boolean

Checks if the list of given material names matches the given base material. Mostly used for configs.

Supports containsCONTAINS:NAME and Regular Expression REGEX:PATTERN formats.

Example:

    XMaterial material = matchXMaterial;
    if (material.isOneOf(plugin.getConfig().getStringList("disabled-items")) return;

The reason that there are tags for CONTAINS and REGEX is for the performance. Although RegEx patterns are cached in this method, please avoid using the REGEX tag if you can use the CONTAINS tag instead. It'll have a huge impact on performance. Please avoid using (capturing groups) there's no use for them in this case. If you want to use groups, use (?: non-capturing groups). It's faster.

Want to learn RegEx? You can mess around in RegExr website.

true if one of the given material names is similar to the base material.
